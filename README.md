# Sistema de Projetos (Delphi)

Sistema desenvolvido em Delphi para gerenciamento de projetos e tarefas.

## Tecnologias

- Delphi (Object Pascal)
- SQL Server
- DevExpress VCL
- Git

## Funcionalidades

- Cadastro de projetos
- Cadastro de colaboradores
- Controle de tarefas

## Banco de dados

O script de criação do banco está disponível em:

ScriptBD/script.sql

## Dependências

Este projeto utiliza componentes da DevExpress (VCL).

Os componentes DevExpress são licenciados comercialmente e **não estão incluídos neste repositório**.  
Para compilar o projeto é necessário possuir uma licença válida da DevExpress instalada no Delphi.
